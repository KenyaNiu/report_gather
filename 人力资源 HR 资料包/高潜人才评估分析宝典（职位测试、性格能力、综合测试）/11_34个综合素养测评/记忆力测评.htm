记忆力测评

　　下面的小测验可以来测查自己的记忆力。仔细阅读每句陈述，看看是否与你的情况符合。

　　 1、我对人名的记忆力很少令我感到尴尬。 是　 否

　　 2、词汇量很丰富，在描述某一个事物时，总能使用大量同义词。 是 否

　　 3、能记住身边同事的太太和小孩的姓名。 是　 否

　　 4、不必寻找档案资料，就能非正式地陈述某种涉及资料的事情。 是 否

　　 5、我很奇怪一些人为什么会提笔忘字。 是　 否

　　 6、不必做一大堆笔记，就能选择性地记住有用的信息。 是　 否

　　 7、对于计划要做的事，我一般都会记住去做。 是　 否

　　 8、我脑中总是有很多的实例和轶事,以便增加说话的趣味和意义。 是 否

　　 9、我经常利用一些技巧,例如联想、谐音等方法来记忆单调的信息,如电话号码、
密码、地址、单词等。 是　 否

　　 10、我一般能记住（或是采用某种笨方法）一些重要日期。 是　 否
